API

To post a new url from command line run the following command in your terminal:

curl -H "Content-Type: application/json" -X POST -d '{"url": "%your url%"}' https://myherokupinger.herokuapp.com/app/new

You should replace "%your url%" with whatever your url is for instance if your url was https://my-header-parser.herokuapp.com you would run the following command:

curl -H "Content-Type: application/json" -X POST -d '{"url": "https://my-header-parser.herokuapp.com"}' https://myherokupinger.herokuapp.com/app/new

If you're url has been posted successfully, you should see something like the following:

Found. Redirecting to /submitted

If you're url has failed and has not been posted, you should see the following error message:

Found. Redirecting to /false-url
